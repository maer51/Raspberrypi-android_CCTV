# Raspberrypi-android CCTV
라즈베리파이 카메라 영상을 핸드폰에서 실시간으로 보여주는 CCTV앱

## 사용한 도구 
* Android Studio (Java)
* Webview API
* Raspberrypi
* (Raspberrypi camera) or usb (camera)

## 주요 기능
* Raspberrypi 카메라 영상을 실시간으로 보여준다
